# SGFood Internal Form Management System

## Project Description
This project is a web-based form management system developed using Laravel.
It supports multi-role workflows including User, Manager, Director, and Admin.

## Features
- User authentication & role-based access
- Form submission and approval workflow
- Manager & Director approval process
- Email notifications
- Admin user management (AJAX + DataTables)

## Technologies
- Laravel 11
- PHP 8.x
- MySQL
- Bootstrap 5
- jQuery & AJAX

## Installation
1. Clone repository
2. Run `composer install`
3. Create `.env` from `.env.example`
4. Run `php artisan key:generate`
5. Run migrations
